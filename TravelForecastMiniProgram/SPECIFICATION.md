# 📖 “游韵华章”智慧文旅小程序开发设计白皮书

## 1. 项目愿景与定位 (Project Vision)

* **项目名称**：游韵华章·智游六盘水 (Smart Tour Liupanshui)
* **核心 Slogan**：**“走遍大地神州，最美多彩贵州”**
* **产品定位**：
* 一款集**“AI数字伴游 + 沉浸式导览 + 红色研学 + 文创电商”**于一体的智慧文旅小程序。
* 以“凉都”六盘水为切入点，用数字化手段展示贵州的多彩文化。


* **目标用户**：来黔游客、研学学生、特产消费者。

## 2. UI 设计规范系统 (Design System)

**核心理念**：将“多彩贵州”的民族元素与“凉都”的清凉感结合，采用**全景沉浸式**设计。

### 2.1 配色体系 (Color Palette)

保持与 Web 端统一，但增加了文化的温度。

* **主色 (Brand Color):** `#2A9D8F` (凉都青) —— 象征避暑、森林、清凉。
* **辅助色 (Secondary):** `#E0F2F1` (云水蓝) —— 用于背景装饰、浅色标签。
* **点缀色 (Accent):** `#FF9F43` (苗绣橙) —— 提取自苗族刺绣，用于**购买按钮、积分、热点**，代表“多彩”。
* **背景色 (Background):** `#F2F5F8` (冷灰白) —— 提升高级感，拒绝死白。
* **特殊纹理**：在卡片背景或 Banner 中，使用 **10% 透明度的苗族银饰纹样** 或 **蜡染冰纹** 作为底纹。

### 2.2 视觉风格 (Visual Style)

* **沉浸式头部 (Immersive Header):** 页面顶部无导航栏边界，大图直接延伸至状态栏。
* **毛玻璃质感 (Glassmorphism):** 悬浮卡片使用 `backdrop-filter: blur(10px)`，营造清透感。
* **3D 图标 (3D Icons):** 金刚区图标采用 3D 渲染风格（如立体的门票、立体的羊肉粉碗），增加趣味性。
* **字体 (Typography):**
* Slogan 专用字：**毛笔书法体**（用于开屏页）。
* 正文：系统默认 (PingFang SC / San Francisco)。



---

## 3. 功能模块详细设计 (Functional Modules)

### 3.1 开屏页 (Splash Screen) —— 文化的初见

* **画面内容**：水墨晕染出的六盘水山水长卷（梅花山/乌蒙大草原）缓缓展开。
* **动画逻辑**：
1. 背景图淡入。
2. 文字 **“走遍大地神州”** 淡入（小字）。
3. 文字 **“最美多彩贵州”** 盖章式放大出现（书法体，朱砂红或渐变金）。
4. 底部出现 Logo：“游韵华章” + “AI 智游凉都”。



### 3.2 首页 (Home) —— 沉浸式种草

* **顶部区域 (Header):**
* 背景：六盘水全景高清图（云海/雪景）。
* 功能：左上角[定位/天气]，右上角[搜索框（半透明）]。
* 文案：大号 Slogan **“赴一场 19°C 的夏天”**。


* **金刚区 (Menu):**
* 悬浮在背景图下半部分。
* 5个入口：**[数字导游]、[红色研学]、[景区预约]、[文创商城]、[AR寻宝]**。


* **AI 入口 (Avatar):**
* 右下角常驻悬浮球。
* 形象：**“黔小游”**（身着苗族/彝族服饰的 Q 版 3D 数字人）。
* 动效：带有呼吸光圈，点击弹出语音对话框。


* **瀑布流推荐:** “当季必玩”、“地道风味”（展示羊肉粉、烙锅）。

### 3.3 智游 (Smart Guide) —— 地图即服务

* **地图引擎**：定制化 Mapbox/腾讯地图，去除杂乱道路，只保留景点。
* **UI 风格**：**手绘风 / 3D 模型风**。
* **交互逻辑**：
* 用户在地图上移动（LBS）。
* 到达“贵州三线建设博物馆”等红色景点时，自动触发数字人讲解。
* **研学任务**：弹出“历史问答卡”，答对获得“黔豆”（积分）。



### 3.4 文创商城 (Shop) —— 带得走的贵州

* **布局**：左侧分类（刺梨/茶叶/火腿/非遗），右侧商品流。
* **特色功能**：
* **一键寄送**：游客在景区买，快递直接发回家。
* **非遗专区**：展示瑞昌剪纸、蜡染、农民画等文创产品。


* **UI 细节**：价格字体放大，使用“苗绣橙”色。

## 4. 技术实现架构 (Technical Architecture)

* **前端框架**：Uni-app (Vue 3)
* **UI 框架**：Tailwind CSS (uni-app版) 或 uView UI 2.0/3.0
* **AI 交互**：
* 语音识别 (ASR) + 语音合成 (TTS)：调用百度/阿里 API。
* 对话大模型 (LLM)：调用你 Web 端后端的 Spring AI 接口。


* **地图**：Mapbox GL JS (Webview方案) 或 微信小程序原生 Map 组件。
