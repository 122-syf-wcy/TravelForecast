## 图片路径说明
所有图片路径统一使用以下格式：
```
/images/图片名称
```

例如：
- 梅花山缩略图：`/images/meihuashan-thumb.jpg`
- 梅花山详情图：`/images/meihuashan-1.jpg`
- 乌蒙大草原缩略图：`/images/wumeng-thumb.jpg`
- 梅花山地图占位图：`/images/meihuashan-map.jpg`

请确保在代码中使用此路径格式引用图片。

## 图片命名规范
每个景区需要提供以下图片：
1. 缩略图：`景区英文名-thumb.jpg`（例如：`meihuashan-thumb.jpg`）
2. 详情图：`景区英文名-1.jpg`、`景区英文名-2.jpg`、`景区英文名-3.jpg`（例如：`meihuashan-1.jpg`）
3. 地图占位图：`景区英文名-map.jpg`（例如：`meihuashan-map.jpg`）

## 地图占位图要求
1. 尺寸：建议600x400像素
2. 内容：景区地图，包含主要景点位置、道路、设施等
3. 格式：JPG格式，清晰可辨识
4. 文件大小：不超过500KB

## 图片上传方式
1. 所有图片直接上传至 `/web/public/images/` 目录
2. 不再使用子文件夹，所有图片平铺在根目录
3. 确保图片符合命名规范、尺寸和质量要求

## 图片尺寸要求
1. 缩略图：300x200像素，用于列表展示
2. 详情图：至少1200x800像素，建议1920x1080像素，用于详情页展示
3. 地图占位图：600x400像素
4. 图片格式：JPG格式，压缩质量80%以上
5. 文件大小：缩略图不超过100KB，详情图不超过2MB，地图占位图不超过500KB
