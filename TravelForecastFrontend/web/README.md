# 游韵华章（六盘水版）

基于Vue3 + TypeScript构建的科技感旅游数据预测平台前端项目，为贵州省六盘水市量身定制。

## 技术栈

- **前端框架**：Vue 3 + TypeScript
- **构建工具**：Vite
- **状态管理**：Pinia
- **UI框架**：Element Plus + Tailwind CSS
- **可视化**：ECharts 5 + Three.js
- **动画效果**：GSAP + Lottie
- **地图服务**：Mapbox GL
- **通信**：Socket.IO
- **HTTP客户端**：Axios

## 功能模块

### 用户端 - 智慧游六盘水

1. **景区探索**：3D地图、热力图、天气信息
2. **客流预测**：未来7天客流、拥挤度预警、最佳游览时间
3. **行程规划**：智能路线规划、时间轴管理、景点间距离
4. **实时服务**：紧急服务点、停车场状态、公交信息

### 商户端 - 景区智慧管理中心

1. **实时监测**：入园人数统计、在园人数分布、客流热力图
2. **数据分析**：游客画像、消费行为预测、停留时间分析
3. **资源管理**：停车场使用率、设施状态、人员调度
4. **评价系统**：情感分析、关键词云、实时反馈处理

### 管理员端 - 旅游数据中枢

1. **平台监控**：实时访问、API性能、异常警报
2. **数据统计**：区域热力图、景区排名、用户行为
3. **系统管理**：用户管理、权限配置、数据备份
4. **决策支持**：预测模型监控、资源分配建议、政策效果模拟
