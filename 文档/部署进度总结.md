- Gateway 端口: 8888

**部署步骤**:
```bash
# 本地构建
cd TravelForecastGateway
mvn clean package -DskipTests

# 上传到服务器
scp target/travel-gateway-*.jar root@39.97.232.141:/opt/travel-gateway.jar

# 启动服务
nohup java -jar /opt/travel-gateway.jar \
  --spring.profiles.active=prod \
  --spring.cloud.nacos.discovery.server-addr=localhost:8848 \
  --spring.data.redis.host=localhost \
  > /opt/gateway.log 2>&1 &
```
